# FINCHEX Official Website

Customized from the Start Bootstrap Grayscale template for GitHub Pages.

## Publish on GitHub Pages
1. Create a GitHub repository, ideally `finchexproductions.github.io`.
2. Upload all files/folders from this website package to the repository root.
3. In GitHub: Settings → Pages → Deploy from a branch → `main` / root.
4. Once live, add the final website URL to Google Search Console, Wikidata (official website), and MusicBrainz (official homepage).

Official identity links and Schema.org structured data are included in `index.html`.
